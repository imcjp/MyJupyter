# MyJupyter
 MyJupyter是一款支持Python和Java的可移动Jupyter软件包。放在U盘里可以轻松实现即插即用，简单快捷的软件开发。
 
 本软件分为5个分卷压缩包，请下载时确保每个压缩包都有下载才能解压。
